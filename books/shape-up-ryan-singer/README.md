# Shape Up → Shape Up Cycles

Adapted from **_Shape Up: Stop Running in Circles and Ship Work that Matters_** by Ryan Singer (Basecamp).

Where `build-sprint` compresses discovery into one week, Shape Up governs the longer rhythm of *how work gets shaped, bet on, and shipped* in six-week cycles. It's the natural companion: sprint to de-risk an idea, then shape and bet it into a cycle.

## Plugin: `shape-up`

| Skill | Purpose |
|-------|---------|
| `shape-up` | Orchestrator — routes between shaping, betting, building, cool-down; enforces the principles |
| `shape-a-pitch` | Set the appetite, rough the solution (breadboard + fat-marker), find rabbit holes and no-gos, write the pitch |
| `bet-the-cycle` | Run the betting table: the five questions, no backlogs, the circuit breaker, kick-off |
| `build-in-cycles` | Hand over responsibility, get one slice done, map scopes (layer cake / iceberg / chowder), hill charts, scope hammering |
| `cool-down` | Between cycles — fix bugs, explore, rest, shape the next round |

**Install:** `/plugin install shape-up@vibe-design`, or open a packaged `.plugin` file in Cowork.

**Use:** say *"let's run Shape Up"* to start with the orchestrator, or name a stage (*"help me shape a pitch"*, *"run our betting table"*, *"map the scopes"*).
